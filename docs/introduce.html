Dev Features

Frontend

• Use Typescript for all JS code.

• Use Scss for styling.

Backend

• Use NestJS for backend

• Use SQL with sqlite

• Server running with ubuntu, on the Naver Cloud Platform.

Auth

• User identified with uuid system v4.

• User's password hashed with the secret salt.

• User can keep login state with passport-jwt module.

Tools

• Docker : Manage linux server and testing APIs.

• Github : manage code log, sharing code with server and deploy frontend.

• VS Code : edit code and debugging.

• Parcel : bundling and build frontend code.

• putty : remote connection for linux server on naver.
